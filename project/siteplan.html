Bergamo Ward Site Plan

Site Name

Bergamo Ward - rionedibergamo.org

This name represents my local ward of the Church of Jesus Christ of Latter-Day Saints

Site Purpose

The site provides a hub for the ward members by providing quick links to useful tools, Church website pages, and ward information (such as talks, baptisms, etc.).

Scenarios

Do I have any talks to give in the coming weeks?

Are there posters for upcoming activities?

Are there any announcements from my quorum or the Relief Society?

Which Church applications could be useful to me?

Color Schema

#1E4881: Header and footer background, headings

#FFFFFF: Header text

#0F0F0F: Body text

#F8F8F8: Body background

#265dA7: Links

Typography

Inter: used for all the typography in the website.

For headings the font weight will be 700, for regular text 400, for links 500
